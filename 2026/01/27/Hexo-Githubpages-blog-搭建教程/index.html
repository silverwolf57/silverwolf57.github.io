Hexo + GitHub Pages 个人博客搭建教程

1. 环境准备

  • Node.js: 运行环境。

  • Git: 代码提交工具。

  • GitHub 账号: 用于托管静态页面。

  1. 在 GitHub 上新建仓库，推荐命名为 用户名.github.io。

YAML

5. 部署上线

PowerShell

6. 更换 Butterfly 主题

由于网络克隆经常失败，建议采用手动模式：

  1. 下载：从 GitHub 下载 Butterfly 主题 ZIP 包。

  2. 安置：解压并重命名为 butterfly，放入 themes/ 文件夹。

  3. 插件：在博客根目录运行：

    npm install hexo-renderer-pug hexo-renderer-stylus --save

  4. 启用：修改站点 _config.yml 中的 theme: butterfly。

7. 一些问题

  • 权限问题：报错 EPERM 时，请以管理员身份运行 PowerShell，且不要在盘符根目录（如 E:\）直接初始化。

  • 脚本限制：若无法运行 hexo，执行 Set-ExecutionPolicy RemoteSigned -Scope CurrentUser。

  • 网络问题：若 git clone 或 npx hexo d 连不上 GitHub，请检查或重置 git 代理：

    git config --global --unset http.proxy

  • 格式问题：_config.yml 修改后若报错 YAML Exception，请检查冒号后面是否漏掉了空格。
